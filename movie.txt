회원관련 : 회원정보, 포인트 사용 내역
영화관련 : 영화정보, 장르, 감독, 배우, 스틸컷
극장관련 : 극장정보, 극장내 상영관, 상영관내 좌석
예매관련 : 상영시간, 표가격, 예매, 예매 좌석

검색
 - 영화 검색 및 선택 -> 날짜 및 상영관 선택 -> 회차 선택 -> 좌석 선택 -> 예매
 - 날짜 검색 및 선택 -> 영화 및 상영관 선택 -> 회차 선택 -> 좌석 선택 -> 예매
 - 상영관 검색 및 선택 -> 날짜 및 영화 선택 -> 회차 선택 -> 좌석 선택 -> 예매

 - 영화 선택 : 현재 상영중인 영화 목록에서 선택
 	검색 조건 : 장르, 영화명, 관람가능 연령대, 주연, 감독 등의 검색 조건
 - 날짜 선택 : 달력으로 예매 날짜를 선택(예매 가능 날짜를 표시)
 - 상영관 선택 : 지역과 멀티플렉스 내의 상영관을 선택해야 함
 	지역 선택 : 도/시/군 등의 지역을 먼저 선택한 후 해당 지역 내의 특정 극장을 선택
	 - 선택한 극장이 단관이 아닐 경우 특정 상영관을 선택할 수 있어야 함
	

한 스크린에서 여러 영화를 상영할 수 있음 <==> 한 영화를 여러 스크린에서 상영할 수 있음
다 대 다의 관계(n:n)

각각의 pk를 이용해 따로 테이블을 작성(각각의 pk를 묶어서 해당 테이블의 pk로 지정)

하나의 예매에 한 영화, 한 회차에 대해서만 예매가 가능
 - 단, 한 회차에 대해 여러 장의 티켓 예매 가능(좌석 선택 가능)
 	다른 영화나 다른 회차 또는 상영관의 경우 다시 예매해야 함
 - 장바구니 기능이 필요 없음
